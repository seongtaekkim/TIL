### package&folder setting

---



- 프로젝트를 진행하다보면, 폴더와 패키지를 구분해야 할 때가 생긴다.
- 폴더와 패키지를 구분하지 못하면, 파일 path에 문제가 생길 수 밖에 없다.
- 이클립스가 자동으로 해주면 좋겠지만, 그러지 못할 경우에 수동으로 설정해야 한다.



프로젝트 우클릭 > properties > java build path > source 위치에서 설정할 수 있다.
